32 bit C kernel to ISO

In 2025, when I was 13, I made this low level script which turns 32 bit protected mode C kernel file into a iso file using GRUB. In this example, I made a fully functional tree chopping game, which I coded myself. When you run this script, it compiles the file, links it, then builds it with GRUB. I made a collection of code for this, and also a small collection of scripts in one ISO file (I called it OllieISO), which you can see of the next slide.
